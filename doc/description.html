Introduction


  • A plugin that helps you standardize submissions.
  • This plugin is an customization version of the source idea plugin git-commit-message-helper,adding a series of personalized configurations.
  • 一个可帮助您标准化提交内容的插件
  • 该插件是源自 git-commit-message-helper 的 dcar 修改版本，添加一系列个性化的配置

How to use



Configuration

如需个性化设置窗口样式，可通过以下路径进入设置菜单：

配置路径：File -> Settings -> Other Settings -> CitCommitMessageHelper

如需自定义提交模板，可通过以下路径进行设置：

配置路径：File -> Settings -> Other Settings -> CitCommitMessageHelper -> Commit Template -> 模板

如需修改提交类型配置，可通过以下路径进行设置：

配置路径：File -> Settings -> Other Settings -> CitCommitMessageHelper -> Commit Template -> 提交类型

如需修改任务平台配置，可通过以下路径进行设置：

配置路径：File -> Settings -> Other Settings -> CitCommitMessageHelper -> Commit Template -> 任务平台
